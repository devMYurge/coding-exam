# 编程面试：Antigravity 动画复刻

## 概述

访问 [antigravity.google](https://antigravity.google)，使用任何技术手段，**像素级复刻**该页面的动画效果。

- 编程语言和框架不限
- 可以使用任何 AI 工具辅助
- 最终产物需要能在浏览器中独立运行

---

## 要求

1. **动画颜色** — 精确还原页面中所有元素的配色方案，包括背景色、粒子颜色、渐变效果等
2. **运动效果** — 还原粒子/元素的运动轨迹和物理行为，包括速度、加速度、弹性、阻尼等
3. **交互效果** — 还原所有用户交互行为（鼠标移动、点击、悬停、滚动等），交互响应需即时自然
4. **粒子运动形变** — 还原粒子在运动过程中的形状变化，包括拉伸、压缩、旋转、缩放等形变效果

> **本题不设分项评分，只有通过与不通过。完成像素级复刻即通过一面。**
